### 3、解决 EasyPOI 模板导出，单元格文本格式导致数字计算问题

```java
/**
 * 将Workbook中所有文本格式的数字单元格转换为数值格式
 * 解决EasyPOI模板导出时，模板单元格为文本格式导致的数字计算问题
 *
 * @param workbook 待处理的工作簿
 */
private void convertTextNumbersToNumeric(Workbook workbook) {
    if (workbook == null) {
        return;
    }

    try {
        DataFormat format = workbook.createDataFormat();
        int convertedCount = 0;

        // 遍历所有sheet
        for (int i = 0; i < workbook.getNumberOfSheets(); i++) {
            Sheet sheet = workbook.getSheetAt(i);
            if (sheet == null) {
                continue;
            }

            // 遍历所有行
            for (Row row : sheet) {
                if (row == null) {
                    continue;
                }

                // 遍历所有单元格
                for (Cell cell : row) {
                    if (cell == null) {
                        continue;
                    }

                    // 只处理字符串类型的单元格
                    if (cell.getCellType() == CellType.STRING) {
                        String cellValue = cell.getStringCellValue();
                        if (StringUtils.isNotBlank(cellValue)) {
                            try {
                                // 尝试将字符串解析为数字
                                double numericValue = Double.parseDouble(cellValue.trim());

                                // 克隆原单元格样式（保留所有原样式属性）
                                CellStyle originalStyle = cell.getCellStyle();
                                CellStyle newStyle = workbook.createCellStyle();
                                newStyle.cloneStyleFrom(originalStyle);

                                // 只修改数据格式为数值格式（保留2位小数）
                                newStyle.setDataFormat(format.getFormat("0.00"));

                                // 设置数值和新样式
                                cell.setCellValue(numericValue);
                                cell.setCellStyle(newStyle);
                                convertedCount++;
                            } catch (NumberFormatException e) {
                                // 不是数字，保持原样
                            }
                        }
                    }
                }
            }
        }
        log.info("已完成文本数字转换，处理了 {} 个Sheet，转换了 {} 个单元格", workbook.getNumberOfSheets(), convertedCount);
    } catch (Exception e) {
        log.warn("文本数字转换过程中发生异常: {}", e.getMessage());
    }
}
```

```java
public static void downloadWorkbook(Workbook workbook, HttpServletRequest request, HttpServletResponse response, String fileName) throws IOException {
    threadLocal.remove();
    String suffix = ".xls";
    if (!(workbook instanceof HSSFWorkbook)) {
        suffix = ".xlsx";
    }

    response.setHeader("content-Type", "application/vnd.ms-excel");

    workbook.write(response.getOutputStream());
```

## 跨域配置

```java
package inks.service.sa.sot.config;

import org.springframework.boot.web.servlet.FilterRegistrationBean;
import org.springframework.context.annotation.Bean;
import org.springframework.context.annotation.Configuration;
import org.springframework.core.Ordered;
import org.springframework.web.cors.CorsConfiguration;
import org.springframework.web.cors.UrlBasedCorsConfigurationSource;
import org.springframework.web.filter.CorsFilter;

@Configuration
public class CorsConfig {

    @Bean
    public FilterRegistrationBean<CorsFilter> corsFilter() {
        // 创建 CORS 配置对象
        CorsConfiguration config = new CorsConfiguration();
        config.setAllowCredentials(true);  // 允许请求携带认证信息（如 Cookies）
        config.addAllowedOriginPattern("*");  // 允许所有来源（域）访问
        config.addAllowedHeader("*");  // 允许所有请求头
        config.addAllowedMethod("*");  // 允许所有请求方法
        config.setMaxAge(1800L); // 解决跨域请求两次，预检请求的有效期，单位为秒

        // 创建基于 URL 的 CORS 配置源，并注册配置
        UrlBasedCorsConfigurationSource source = new UrlBasedCorsConfigurationSource();
        source.registerCorsConfiguration("/**", config);

        // 创建 CORS 过滤器的注册对象
        FilterRegistrationBean<CorsFilter> bean = new FilterRegistrationBean<>(new CorsFilter(source));
        bean.setOrder(Ordered.HIGHEST_PRECEDENCE); // 设置过滤器的执行顺序为最高，确保优先执行
        return bean;
    }
}
```

## @RequestParam 注解

- `defaultValue` 使用时可省略 `required = false`
- 简单参数：写 `@RequestParam` 更规范
- JSON 请求体：必须用 `@RequestBody`

```java
// 参数名映射
@RequestParam("git_url") String gitUrl

// 默认值
@RequestParam(defaultValue = "master") String branch

// required 是否必须
@RequestParam(required = false)

// 可替换写法
@RequestParam(required = false, defaultValue = "master") String branch
// 等价于
@RequestParam(defaultValue = "master") String branch
```

## JAVA 函数式编程（Stream）写法举例

### 基础语法

**1️⃣ 创建流**

```java
list.stream()
array.stream()
Stream.of(1, 2, 3)
```

**2️⃣ 中间操作（不产生结果）**

| 操作 | 说明 |
|------|------|
| `filter` | 过滤 `.filter(x -> x > 10)` |
| `map` | 映射（转换）`.map(x -> x.getName())` 把对象转换成别的对象 |
| `flatMap` | 扁平化，常用于 `List<List<T>>` → `List<T>` |
| `distinct` | 去重 |
| `sorted` | 排序 `.sorted(Comparator.comparing(User::getAge))` |
| `limit` | 截断 `.limit(5)` |
| `skip` | 跳过 `.skip(10)` |

**3️⃣ 终止操作（产生结果）**

| 操作 | 说明 |
|------|------|
| `collect` | `.collect(Collectors.toList())` / `.toSet()` / `.toMap()` / `.groupingBy()` / `.partitioningBy()` |
| `forEach` | `.forEach(System.out::println)` |
| `count` | `.count()` |
| `anyMatch` | 只要有一个满足就 true |
| `allMatch` | 全部满足才 true |
| `noneMatch` | 全不满足才 true |
| `findFirst` | 返回 Optional |
| `findAny` | 返回 Optional |
| `max` / `min` | `.max(Comparator.comparing(User::getAge))` |

## 线程的基础使用方式

### 1、多线程日志打印

> 下载文件需要5秒，日志打印需要1秒，要求日志打印在下载完成之前完成

```java
public static void main(String[] args) {
    new Thread(() -> {
        download();
    }).start();

    printLog();
}

public static void download() {
    try {
        System.out.println("开始下载");
        Thread.sleep(5000);
        System.out.println("下载完成");
    } catch (Exception e) { }
}

public static void printLog() {
    System.out.println("打印日志");
}
```

### 2、多线程下载

> 单线程：每个任务3秒，串行共9秒
> 多线程：每个任务3秒，并行共3秒

```java
public static void main(String[] args) throws Exception {
    // 单线程下载，每个任务需要3秒钟，串行执行总共需要9秒钟
    task("任务1");
    task("任务2");
    task("任务3");

    System.out.println("-----------------------------多线程开始-----------------------------");

    // 多线程下载，每个任务需要3秒钟，并行执行总共只需要3秒钟
    new Thread(() -> task2("任务1")).start();
    new Thread(() -> task2("任务2")).start();
    new Thread(() -> task2("任务3")).start();
}

public static void task(String name) throws Exception {
    System.out.println(name + "开始");
    Thread.sleep(3000);
    System.out.println(name + "结束");
}

public static void task2(String name) {
    try {
        System.out.println(name + "开始");
        Thread.sleep(3000);
        System.out.println(name + "结束");
    } catch (Exception e) { }
}
```

### 3、线程池的基础说明

```java
public static void main(String[] args) {
    // 创建线程池，创建一个固定3个线程的线程池
    ExecutorService pool = Executors.newFixedThreadPool(3);

    // 总共有5个任务
    for (int i = 1; i <= 5; i++) {
        // Lambda 里只能使用最终变量
        int num = i;
        // 把任务交给线程池执行
        pool.execute(() -> {
            System.out.println(
                    Thread.currentThread().getName()
                            + " 执行任务 "
                            + num);
            try {
                Thread.sleep(2000);
            } catch (Exception e) { }
        });
    }
    // 关闭线程，线程池不再接收新任务
    pool.shutdown();
}
```

## Map 的常见方法

### 1、merge

```java
Map<String, Integer> userCountMap = new HashMap<>();
userCountMap.merge("U001", 2, Integer::sum);
userCountMap.merge("U001", 3, Integer::sum);
userCountMap.merge("U001", 4, Integer::sum);
// U001 = 2 → 2+3=5 → 5+4=9
// { "U001": 9 }

// 等价写法
userCountMap.put(
    userId,
    userCountMap.getOrDefault(userId, 0) + hitCount
);
```

### 2、put() 和 putIfAbsent()

- `put()`：如果 key 存在则覆盖
- `putIfAbsent()`：如果 key 不存在则新增，存在则保留不覆盖

### 3、getOrDefault()

不存在返回默认值：

```java
Integer value = map.getOrDefault("B", 0);
```

### 4、containsKey() 和 containsValue()

```java
if (map.containsKey("A")) { /* 存在 */ }

// containsValue 效率较低，需要遍历
map.containsValue(10);
```

### 5、computeIfAbsent 和 computeIfPresent

```java
// 分组：{ IT=[张三] }
Map<String, List<String>> map = new HashMap<>();
map.computeIfAbsent("IT", k -> new ArrayList<>()).add("张三");

// 如果存在则计算：A = 原值 + 10
map.computeIfPresent("A", (k, v) -> v + 10);
```

### 6、forEach() 遍历和 entrySet() 遍历

```java
map.forEach((k, v) -> {
    System.out.println(k + "=" + v);
});

// entrySet 效率最高
for (Map.Entry<String, Integer> entry : map.entrySet()) {
    System.out.println(entry.getKey() + "=" + entry.getValue());
}
```
