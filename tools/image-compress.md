# 📦 图片容量压缩

比[图片格式转换](/tools/image-convert)更聚焦，专门解决「体积太大」的问题：

- **按质量**：直接指定 0 ~ 100%，即压即出
- **按目标大小**：给一个目标 KB（比如"要塞进 200KB 内"），
  工具会用**二分法在 8 轮内**自动找到最接近但不超过目标的质量
- **最长边限制**：顺便缩尺寸（4032 的原图 → 2000 边够用）
- 显示**每张体积对比**和**总节省百分比**

::: tip 什么时候用哪个
- 想控制清晰度 → 按质量（照片一般 70~85% 视觉无损）
- 想上传附件、微信头像、公众号封面等有硬性大小限制 → 按目标大小
- WebP 通常比同质量 JPG 再小 25% 以上，浏览器都支持
:::
