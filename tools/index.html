🧰 开发工具 ​

平时写代码时用得上的小工具，纯前端实现、不发请求、放心用。 后续会陆续加：URL 编解码、Base64、UUID、正则测试、SQL 格式化、颜色转换、Diff……

🌐
时区转换时间
任意城市 / UTC 偏移之间互转，也能看当前各时区的时间
⏱️
时间戳转换时间
秒 / 毫秒时间戳与日期字符串互转，支持自定义时区和格式
🧾
JSON 格式化文本
容错解析 —— 单引号、注释、尾逗号、无引号 key 都能处理
🖼️
图片 ↔ Base64编码
拖拽 / 粘贴图片转 Base64；也能把 Base64 还原成图片并下载

想加新工具？

在 .vitepress/theme/components/ToolGrid.vue 的 tools 数组里补一条，再对应新建一个 .vue 组件和 tools/xxx.md 页面即可。
