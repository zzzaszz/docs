# 🎨 图片格式转换

PNG / JPG / WebP / BMP 互转，支持：

- 批量拖入多张一起转
- 调质量（0.1 ~ 1.0，对 JPG / WebP 有效）
- 缩放比例（0.1 ~ 2.0）
- 单张下载或全部下载
- 显示前后体积对比

::: tip 提示
- **JPG / BMP 不支持透明**，转换时会自动铺白底。
- **WebP 通常最小**，同质量下比 JPG 再省 25% 以上，现代浏览器都支持。
- 图片全程在浏览器本地处理、不上传。
:::
